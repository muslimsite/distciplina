# Дисциплина от А до Я

Лендинг интенсива «Дисциплина от А до Я» (Сёстрам · Солиха Умм Муса).
Статический сайт — чистый HTML/CSS/JS, без сборки и зависимостей.

## Файлы
- `index.html` — вся страница (стили и скрипты внутри). Аватар Солихи встроен прямо в файл.
- `assets/title-cut-main.png` — заголовок «Дисциплина от А до Я» (надпись с прозрачным фоном). Нужен для первого экрана; если файл не загрузится, вместо него автоматически покажется текстовый заголовок.
- `assets/avatar-cut.png` — запасная копия силуэта-аватара (в index.html он уже встроен, но пусть лежит).
- `.nojekyll` — отключает обработку Jekyll на GitHub Pages

## Что обновлено в этой версии
- Новый дизайн: заголовок-картинка, мини-карточки (старт/длительность/доступ/уроки), блок «Для кого», «В результате», кнопка оплаты, «Служба поддержки».
- Сохранены прошлые правки: увеличенный аватар в круге (встроен в HTML), анимации появления и золотые блики, навигация (меню + бургер на мобильном), контрастные карточки.

## Как опубликовать на GitHub Pages
1. Создайте репозиторий и загрузите в него все файлы (index.html, папку assets, .nojekyll).
2. В репозитории: Settings → Pages.
3. Source: «Deploy from a branch», ветка `main`, папка `/ (root)` → Save.
4. Через 1–2 минуты сайт будет доступен по адресу
   index.html самодостаточен — аватар встроен прямо в файл, папка assets больше не обязательна.

Кнопка оплаты сейчас в демо-режиме (показывает заглушку). Когда будет платёжка — подключим на кнопку «Оплатить».
